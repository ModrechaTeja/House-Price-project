House Price Prediction Web Application

Built a machine learning model to predict residential house prices using the Ames Housing dataset.
Performed data cleaning, feature selection, and preprocessing.
Trained a Random Forest Regressor and evaluated performance using regression metrics.
Developed a Flask-based web application for real-time price prediction.
Integrated the trained model with a user-friendly web interface.

dataset--https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data?select=train.csv
